Active Users List
=================
わんこめに簡単導入できる「現在配信に参加しているユーザー一覧」テンプレートです。
OBS のブラウザソースに読み込むだけで、コメントしたリスナーが自動で表示されます。
プログラミング知識は不要です。

------------------------------------------------------------
■ 使い方（非エンジニア向け）
------------------------------------------------------------
1. GitHub の Releases ページから最新バージョンを開く
2. active_user_list-xxx.zip をダウンロード
3. ZIP を解凍
4. OBS を開き、「ソース追加 → ブラウザ」を選択
5. 「ローカルファイルを使用する」にチェック
6. 解凍フォルダの index.html を指定
7. 配信を開始するとコメントした順にリスナーが表示されます

------------------------------------------------------------
■ config.json の設定（任意）
------------------------------------------------------------
フォルダ内の config.json をメモ帳で開くと、挙動を変更できます。

userLifetimeSeconds: 名前を残す秒数
highlightSeconds: 新規参加時のハイライト秒数
maxActiveUsers: 同時に表示する最大人数

例:
{
  "userLifetimeSeconds": 90,
  "highlightSeconds": 5,
  "maxActiveUsers": 20
}

------------------------------------------------------------
■ 注意点
------------------------------------------------------------
- YouTube APIキー不要。OBSブラウザだけで動きます。
- CSSファイル(style.css)でデザインを自由に変更できます。
- コメント量が多いと maxActiveUsers を超えた分は古い順に消えます。
